# sturdy-tribble-chatbot

A website featuring an interactive idea exploration tool and content management via Sparktype CMS.

## Features

- **Idea Explorer**: Interactive AI-powered mindmap tool for exploring concepts
- **Sparktype CMS**: Browser-based content management for pages
- **GitHub Pages**: Automatic deployment on push to main branch
- **Mobile-Friendly**: Works on all devices

## Quick Start

Visit: [pennyphilosopher.com](https://pennyphilosopher.com)

## Content Management

This site uses [Sparktype](https://app.sparktype.org) for content management. See [SPARKTYPE_GUIDE.md](SPARKTYPE_GUIDE.md) for detailed instructions.

### Quick Workflow:

1. Edit content at [app.sparktype.org](https://app.sparktype.org)
2. Export your site
3. Commit to `/pages` directory
4. Push to deploy automatically
